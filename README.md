## Usage

```swift
import SimulatorCheck

if SimulatorCheck.isSimulator {
    // Do something
}
```

## Installation

**SimulatorCheck** is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'SimulatorCheck'
```
